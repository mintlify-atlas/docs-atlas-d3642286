## What is TokenStash?

TokenStash is a Claude Code plugin that caches project context locally — so Claude starts every session already knowing your codebase. No re-reading files. No re-explaining architecture. Just start coding.

## Quick Example

```bash
# Install TokenStash
/plugin marketplace add abhinavdubeyad9/tokenstash
/plugin install tokenstash@tokenstash

# Start a new session — your stash is loaded automatically
# Claude already knows your project structure, decisions, and key files

# Save your session summary
/stash-save

# Scan the entire repo
/stash-scan

# Force re-scan everything
/stash-scan --force

# Clear the stash
/stash-clear
```

## What Gets Stashed

TokenStash maintains multiple context files:

- **architecture.md** — High-level codebase overview, structure, patterns, key components
- **decisions.md** — Decision log — what was decided each session and why
- **rules.md** — Project-specific rules and conventions
- **files/\*.md** — Per-file summaries (3-5 lines each)
